# Djule Examples

These files are the first syntax fixtures for Djule v1.

- `01_simple_page.djule`: plain page with one prop
- `02_component_import.djule`: imported components in markup
- `03_children.djule`: reserved `children` prop
- `04_logic_above_return.djule`: Python logic before markup
- `05_embedded_if_else.djule`: embedded block-style conditional
- `06_embedded_for.djule`: embedded block-style loop
- `07_nested_components.djule`: nested composition and prop drilling
- `08_django_request_props.djule`: root component receiving Django request data
